# Andes Solar Hash — Sitio estático

Versión estática del sitio web (HTML/CSS/JS) para despliegue en hosting estático (GitHub Pages, Netlify, Vercel, etc.).

Generado desde el proyecto [andessolarhash](https://github.com/1511170/andessolarhash) con `npm run build` (adapter-static).

## Desplegar

- **GitHub Pages:** sube este repo y activa Pages en la rama `main` (carpeta raíz).
- **Netlify / Vercel:** conecta el repo y usa la raíz como directorio de publicación.

## Actualizar

Desde el repo fuente:

```bash
cd andessolarhash
npm run build
# Copiar contenido de build/ a andessolarhash-static y hacer commit
```
